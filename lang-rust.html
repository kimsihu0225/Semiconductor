Programming Language

Rust

Rust는 성능과 메모리 안정성을 동시에 잡는 시스템 프로그래밍 언어로, C/C++ 대안으로 자주 언급됩니다.

특징

안전성과 성능을 동시에 노리는 언어입니다.

주요 직무

시스템 개발, 툴 개발, 성능 민감 서비스

주요 툴

VS Code, cargo, rustc

학습 포인트

소유권, 빌림, 수명 개념이 핵심입니다.

장점과 주요 활용

항목정리
무엇에 특화되어 있나시스템 수준 성능과 메모리 안전성을 동시에 요구하는 환경에 강합니다.
장점런타임 에러를 줄이고 성능도 높은 편이라 안정적인 저수준 개발에 유리합니다.
반도체 외 주요 활용시스템 도구, 브라우저 엔진 일부, 보안 소프트웨어, CLI 툴, 고성능 백엔드에 쓰입니다.
