증착 공정 (Deposition)

기존 사이트의 증착 공정 관련 내용을 한 페이지로 모았습니다.

목적: 절연막 형성과 표면 보호.

원리: 실리콘 표면을 산소 또는 수증기와 반응시켜 SiO2를 성장시킵니다.

주요 장비: Furnace, oxidation tube.

주요 재료: O2, H2O, 실리콘 웨이퍼.

대표 defect: 박막 두께 불균일, particle, pinhole.

중요 포인트: 막질 균일성과 산화 온도 제어가 핵심입니다.

Coat Expose Develop

목적: 회로 패턴을 웨이퍼 위에 전사합니다.

원리: 포토레지스트 도포 후 노광과 현상을 통해 원하는 패턴만 남깁니다.

주요 장비: Track, Scanner, Coater/Developer.

주요 재료: Photoresist, developer, mask/reticle.

대표 defect: misalignment, footing, scum, collapse.

중요 포인트: CD, overlay, focus, dose 관리가 현업에서 매우 중요합니다.

Step-by-step

1. 포토레지스트 도포 → 2. Soft bake → 3. 노광 → 4. 현상 → 5. 후속 공정 연결

Mask Plasma Remove

목적: 필요한 부분만 선택적으로 제거해 패턴을 형성합니다.

원리: 플라즈마 또는 화학 용액 반응을 이용해 막을 제거합니다.

주요 장비: Dry etcher, wet bench.

주요 재료: CF4, Cl2, BCl3, 세정액.

대표 defect: over-etch, residue, profile imbalance, damage.

중요 포인트: 선택비, 이방성, profile control이 핵심입니다.

Step-by-step

1. Mask 준비 → 2. Plasma 형성 → 3. 반응종 충돌/제거 → 4. endpoint 확인 → 5. residue 세정

Gas In Reaction Film

목적: 절연막, 도전막, barrier막 등을 웨이퍼 위에 형성합니다.

원리: 기상 반응 또는 물리적 전달로 박막을 쌓습니다.

주요 장비: CVD, ALD, PVD 장비.

주요 재료: SiH4, NH3, TEOS, TiN, W, Cu.

대표 defect: 막질 불균일, particle, poor step coverage.

중요 포인트: 두께, 균일도, 막질, conformality가 중요합니다.

Step-by-step

1. 가스 공급 → 2. 반응 생성 → 3. 표면 흡착 및 성장 → 4. purge → 5. 두께 검증

📌 핵심 정리

  • 증착 공정 (Deposition)은 웨이퍼 위에 절연막, 도전막, 배리어막을 쌓는 공정입니다.
  • CVD (Chemical Vapor Deposition)는 기체 반응으로 막을 형성합니다.
  • PVD (Physical Vapor Deposition)는 물리적으로 재료를 날려 막을 입힙니다.
  • ALD (Atomic Layer Deposition)는 원자층 단위로 얇고 균일한 막을 만들 수 있습니다.

📌 쉽게 이해하기

  • 증착은 웨이퍼 위에 아주 얇은 페인트 층을 균일하게 입히는 과정과 비슷합니다.
  • 구조가 복잡할수록 구석까지 고르게 덮는 능력 (Step Coverage)이 중요합니다.
  • 막이 너무 두껍거나 얇으면 전기 특성과 후속 공정이 흔들립니다.

🔧 핵심 장비

  • 화학기상증착 장비 (CVD System): 반응 가스로 박막을 형성합니다.
  • 원자층증착 장비 (ALD System): 전구체를 번갈아 공급해 원자층 단위 막을 만듭니다.

🚀 미래 장비 (10년 후)

차세대 원자층 증착 클러스터 (Next-Gen ALD Cluster Tool)

차세대 ALD 클러스터는 복잡한 3D 구조 위에도 균일한 막을 만들기 위해 더 정밀한 전구체 공급과 퍼지 제어를 사용합니다. 공정 중 센서로 막 성장 상태를 추적하고, 필요한 경우 레시피를 미세 조정하는 방향으로 발전합니다. GAA (Gate-All-Around), 3D NAND, 첨단 패키징에서 중요성이 커집니다.

⚡ 현재 장비와 차이점

  • 막 두께 제어가 더 정밀해져 원자층 단위 균일성이 향상됩니다.
  • 복잡한 3D 구조 내부까지 덮는 능력이 강화됩니다.
  • 전구체 사용 효율과 배기 관리가 개선됩니다.
  • 실시간 센서 기반 막질 모니터링 비중이 커집니다.
